

NFT маркетплейс — фронтенд приложение без бэкенда.



- Регистрация и авторизация пользователей
- Гостевой доступ (только просмотр)
- Каталог NFT с поиском и фильтрацией по редкости
- Детальная страница NFT с описанием и комментариями
- Голосование за NFT (upvote / downvote)
- Покупка NFT в личную коллекцию
- Создание (минтинг) новых NFT
- Профиль пользователя с коллекцией



- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Данные хранятся в памяти (симуляция бэкенда)
